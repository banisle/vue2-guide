Param

Method

Event

 

 

UserLangv1.3 higher

브라우저 언어와 상관없이 사용하는 다국어 리소스 중 특정 언어 리소스를 사용하고자 할 경우 사용합니다.

 

2개 국어 이상의 다국어 지원 크로스에디터를 사용하는 경우에만 사용할 수 있습니다.

기본 언어에서 다국어 리소스를 추가할 경우 언어별 구매 금액의 10% 비용이 추가됩니다.

 

Parameters

설정

ㆍ한국어 : kor
ㆍ영어 : enu
ㆍ일본어 : jpn
ㆍ중국어 간체 : chs
ㆍ중국어 번체 : cht
ㆍ베트남어 : vit
ㆍ인도네시아어 : ind
ㆍ자동설정 : auto
  (브라우저 언어에 해당하는 언어 리소스가 없을 경우 기본 언어로 설정됨)

설정 안 함
