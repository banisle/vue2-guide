Param

Method

Event

 

 

Templatev2.0.3.06 or higher

에디터에서 템플릿 메뉴의 목록을 설정합니다.

 

HTML 파일 경로는 URL 절대 경로를 사용합니다.

에디터가 실행되는 서버의 도메인 정보와 다른 도메인의 파일 경로는 사용할 수 없습니다.

 

Parameters

설정

템플릿 메뉴에 표시되는 문서명, 경로, 인코딩 정보 설정

- title : "Selectbox에 보여질 문서제목"

- url : "문서의 Full경로 또는 절대경로"

- charset : "문서파일의 인코딩, 생략가능"

설정 안 함

기본 값

 

Sample Codes

var CrossEditor = new NamoSE("test");
CrossEditor.params.Template = [ 
         { 
                title : "템플릿1",
                url : "/template/t1.html",
                charset : "utf-8"
         }, 
         {
                title : "템플릿2",
                url : "http://namoeditor.co.kr/template/t2.html",
                charset : "euc-kr"
         } 
];
CrossEditor.EditorStart();
