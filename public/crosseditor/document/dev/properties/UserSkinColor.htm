Method

Event

 

 

UserSkinColor

에디터 UI에 각 라인별 색상을 설정합니다.

 

UserSkinColor 값은 외각선, 내각선, 폰트 색상, 툴바 배경색으로 구성돼 있습니다. 값 입력 시 네 가지의 값을 모두 입력해야 합니다. 각각의 값은 구분자(|)로 구분하여 입력해야 합니다.

 

Parameters

설정

UI에 라인별 색상 설정

[ 입력 순서 ] 외각선 | 내각선 | 폰트색상 | 툴바 백그라운드 색상

설정 안 함

기본 값

 

Sample Codes

var CrossEditor = new NamoSE("test");
CrossEditor.params.UserSkinColor = "yellow|blue|black|white";
CrossEditor.EditorStart();
