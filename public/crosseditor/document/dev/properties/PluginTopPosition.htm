PluginTopPositionv3.5.1.04 or higher

iframe 안에 에디터 생성 시 에디터 플러그인 창 높이보다 iframe 높이가 낮은 경우 가려지지 않도록 설정합니다.

설정 시 window.top에 플러그인 창이 생성됩니다.

 

Parameters

TRUE

설정

FALSE

설정 안 함 (기본 값)

 

Sample Codes

var CrossEditor = new NamoSE("test");
CrossEditor.params.PluginTopPosition = true;
CrossEditor.EditorStart();
